Coding the Program - Part ii


Using Conditions to Filter Our User Input

In this part of the code we will be using if statements to filter the users inputs and call the correct calculation methods that we made above. For Example: if the user inputs “A” for addition we will call the addition method that we have made, and vice versa.

This part of the code is a little complicated and can be hard to wrap our had around. As a result, I have written some pseudo code to simplify the code.

Pseudo Code:


While this statement is true:

    Ask user to input a letter corresponding to a calculation.

    If the users input is valid:
        
        #Nested If statment
        If user input is “A”:
            Run Addition method.

        If user input is “B”:
            Run Subtraction method.
        
        If user input is “C”:
            Run Multiplication method.

        If user input is “D”:
            Run Division method.

        If user input is “E”:
            Run Floor Division method.

        If user input is “F”:
            Run Modulus method.

        If user input is “G”:
            Run Area Of A Circle method.

        If user input is “H”:
            Run Perimeter Of A Circle method.

        If user input is “I”:
            Run Exponentiation method.

        If user input is “J”:
            Run Celsius to Fahrenheit method.

        If user input is “K”:
            Run Fahrenheit to Celsius method.

If user input is “X”:
            Break the While loop.

        
    Else:
        Tell user that there input is invalid.  
    
    Continue Back to the start of the while loop.

Now we can use our pseudo code to have a better idea of our to code this part of our project. We will be using while loops and several If statements.

Input Manager Code:


#This statement will always run because the conditions is set to true. Using ‘continue’ will only exit this loop

while True:
  
  choice = input("Enter choice(A, B, C, D, E, F, G, H, I, J, K, X): ")
  choice = choice.upper()

# This if statement checks to see if the users inputs are correct
  if choice in ('A', 'B', 'C', 'D', 'E', 'F', 'G', 'H', 'I', 'J', 'K', 'X'):
   
    if choice == 'A':
      print("You Have Selected Addition")
      num1 = float(input("Enter The First Number You Want To Add: "))
      num2 = float(input("Enter The Second Number You Want To Add: "))
      print(str(num1)+ " + " + str(num2)  +  " = " +  str(add(num1, num2)))
  
    elif choice == 'B':
      print("You Have Selected  Subtraction")
      num1 = float(input("Enter The First Number You Want To Subtract: "))
      num2 = float(input("Enter The Second Number You Want To Subtract: "))
      print(str(num1) + " - " + str(num2)  +  " = " +  str(subtract(num1, num2)))
  
    elif choice == 'C':
      print("You Have Selected Multiplication")
      num1 = float(input("Enter The First Number You Want To Multiply: "))
      num2 = float(input("Enter The Second Number You Want To Multiply: "))
      print(str(num1) + " x " + str(num2)  +  " = " +  str(multiply(num1, num2)))
      
    elif choice == 'D':
      print("You Have Selected Division")
      num1 = float(input("Enter The First Number To Be Divide: "))
      num2 = float(input("Enter The Second Number You Want To Divide By: "))
      print(str(num1) + " / " + str(num2)  +  " = " +  str(divide(num1, num2)))
      
    elif choice == 'E':
      print("You Have Selected Floor Division")
      num1 = float(input("Enter The First Number You Want To Floor Divide: "))
      num2 = float(input("Enter The Second Number You Want To Floor Divide By: "))
      print(str(num1) + " // " + str(num2)  +  " = " +  str(floorDivision(num1, num2)))
      
    elif choice == 'F':
      print("You Have Selected Modulus")
      num1 = float(input("Enter first number: "))
      num2 = float(input("Enter second number: "))
      print(str(num1) + " Modulus of " + str(num2)  +  " = " +  str(modulus(num1, num2)))
      
    elif choice == 'G':
      print("You Have Selected Area Of A Circle")
      num1 = float(input("Enter the Radios of the Circle: "))
      print("The area of a circle with a radios of " + str(num1) + " is: " +  str(circleArea(num1)))
      
    elif choice == 'H':
      print("You Have Selected Perimeter Of A Circle")
      num1 = float(input("Enter the Radios of the Circle: "))
      print("The perimeter of a circle with a radios of " + str(num1) + " is: " + str(circlePerim(num1)))
      
    elif choice == 'I':
      print("You Have Selected Exponentiation")
      num1 = float(input("Enter a number "))
      num2 = float(input("Enter a power"))
      print(str(num1) + " to the power of " + str(num2)  +  " = " +  str(powers(num1, num2)))
      
    elif choice == 'J':
      print("You Have Selected To Convert Celsius to Fahrenheit")
      num1 = float(input("Enter in a  Celsius amount: "))
      print(str(num1) + " Celsius to Fahrenheit is: " +  str(celToFahren(num1)))
      
    elif choice == 'K':
      print("You Have Selected To Convert Fahrenheit to Celsius")
      num1 = float(input("Enter in a  Fahrenheit amount: "))
      print(num1, "=", fahrenToCel(num1))
      print(str(num1) + " Fahrenhite to Celsius is: " +  str(fahrenToCel(num1)))
    
    elif choice == 'X':
      break
        
    continue
  else:
    print("Invalid Input")
        
print("Goodbye")
