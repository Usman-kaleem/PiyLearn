Numbers


We have seen different number datatypes, now we will see what these different number types are and how they are different to each other.

There are three main number types in python.

Int

An Int is short for Integer, this is a whole number that can either be positive or negative. For example, 1, 10, -21 and -23329 but not 21.32 as this is not a whole number.

Code Example:


intNumber1 = 21
intNumber2 = -842

print(type(intNumber1))
print(type(intNumber2))

Code Output:


# --Output--

<class 'int'>
<class 'int'>

Float

A float is what is represented by a real number, they are numbers that use a decimal point to show fractional values. These can be both positive or negative. For example: 10.5, - 10.5 and 232.434.

With floats we can even use a scientific notation using ‘E’ or ‘e’. This indicates powers of 10 and is used for very large numbers. For example: 23.43e14 this is the same as 23.43 x 10(to the power of 14) = 234,300,000,000

Code Example:


floatNumber1 = 21.54
floatNumber2 = -842.002
floatNumber3 = 23.4E16

print(type(floatNumber1))
print(type(floatNumber2))
print(type(floatNumber3))

Code Output:


# --Output--

&itclass 'float'>
&itclass 'float'>
&itclass 'float'>

Complex

A complex number is a float but with a ‘j’ at the end. This ‘j’ represents an imaginary number. For example: 2.501j, and 23.43e14j

Code Example:


complexNumber1 = 21.54+3j
complexNumber2 = -842.002j
complexNumber3 = 23.4E16j

print(type(complexNumber1))
print(type(complexNumber2))
print(type(complexNumber3))

Code Output:


# --Output--

&itclass 'complex'>
&itclass 'complex'>
&itclass 'complex'>

Number Type Conversions

Say we want to convert a float into a int or a int into a complex and vice versa. We can do this by using these methods that are built into python. This is known as casting.

int() – Method that converts different number types to an integer

float() - Method that converts different number types to an float

complex() - Method that converts different number types to an complex

Code Example:


intNumber = 21
floatNumber = 21.54
complexNumber = 21.54+7j

print(type(intNumber))
print(type(floatNumber))
print(type(complexNumber))

intNumberConverted = float(intNumber)
floatNumberConverted = complex(complexNumber)
complexNumberConverted = int(floatNumber)

print("")
print(intNumberConverted)
print(floatNumberConverted)
print(complexNumberConverted)

print("")
print(type(intNumberConverted))
print(type(floatNumberConverted))
print(type(complexNumberConverted))

Code Output:


# --Output--

&itclass 'int'>
&itclass 'float'>
&itclass 'complex'>

21.0
(21.54+7j)
21

&itclass 'float'>
&itclass 'complex'>
&itclass 'int'>
