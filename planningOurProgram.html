Planning our Program


Using everything we learned we will be making a basic console application. This means we will be interacting with our program via the console. The application we will be building is a calculator.

When making any program it is important to plan before developing.

Program Requirments

Lets see what the requirements for our calculator will be.

- Basic addition, subtraction, division, and multiplication
- Ability to do floor division and modulus
- A powers calculator
- Circle area and perimeter calculator
- Temperature converter between F and C


Pseudo Code

Now that we know what our system should include, we are now able to start planning our programs. We should plan out our program before making it so that we have a better understand of what we are doing.

One way to plan out our program is to use Pseudo code. Pseudo Code is basically writing our code in plain English. This makes it very easy to understand what our code will do.

No let us make some pseudo code for our program. Remember this is not real code.

Pseudo Code:

Division Method

Exponentiation Method

Celsius to Fahrenheit Converter Method
