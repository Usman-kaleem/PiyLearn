Basic Math


Lets look at how we can use basic operations in python to do some basic maths.

Addition, Subtraction, Multiplication and Division

Code Example:


a = 5
b = 10
c = 55
d = 1065

print(a + d) # Addition
print(b - c) # Subtraction
print(c * b) # Multiplication
print(d / a) # Division

Code Output:


# --Output--

1070
-45
550
213.0

Modulus

The Modulus works like how division works but instead of showing a division it only shows the remainder.

Code Example:


a = 54
b = 5
c = 7

print(a % b)# 5 goes into 50 ten times and leaves us with 4 remaining as 4 is not devisable by 5
print(a % c)

Code Output:


# --Output--

4
5

Exponentiation

This is multiplying something by the power of something else. For Example 3³ which is the same as 3 x 3 x 3 = 27.

Code Example:


a = 5
b = 3
c = 2

print(a ** b) 
print(b ** c) 
print(a ** c)

Code Output:


# --Output--

125
9
25

Floor division

Floor division works similar to how normal division works however it will round down the result to the nearest whole number.

Code Example:


a = 20.5
b = 2

print(a / b) # Standard division this can be a decimal number
print(a // b) # Floor Division this will always be a whole number that is rounded down

Code Output:


# --Output--

10.25
10.0
