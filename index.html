Indentation


In Python we use Indentation to signify a block of code. Indentation is the blank space in front of a line of code, this is done by pressing the “Tab” button on the keyboard.


Code Example:


X = 1
Y = 3

If X > Y 
    Print(“X is larger then Y”) 
    Print(“Y is not larger then X”)

As you see from the example above we have a block of code that gets executed after the If statement.

In other programming languages Indentation is only used to make the code easier to read. But in Python an error would occur.
