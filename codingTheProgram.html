Coding the Program


Now what we have our pseudo code, we can now use it to help use start coding our Program. We will do this in steps.

Making our Methods

Let us start by making our methods, these methods will be responsible for each of our calculations

Code of our Methods:


#Method for Addition
def add(num1, num2):
    return num1 + num2

#Method for Subtraction
def subtract(num1, num2):
    return num1 - num2

#Method for multiplication 
def multiply(num1, num2):
    return num1 * num2

#Method for divition
def divide(num1, num2):
    return num1 / num2

#Method for modulus
def modulus(num1, num2):
  return num1 % num2

#method for floor Division  
def floorDivision(num1, num2):
  return num1 // num2

#Method for exponential calculation
def powers(num1, num2):
    return num1 ** num2

The Methods We made above were relatively simple, now let us move on to the calculation functions that are a bit more complex.

Making our More Complex Methods:


#method for calculating the area of a circle
def circleArea(radi):
  pi = 3.14
  return pi * (radi * radi);
  
# method for calculating the perimeter of a circle
def circlePerim(radi):
  pi = 3.14
  return 2 * pi * radi;
  
#method to convert Fahrenheit to Celsius 
def fahrenToCel(fahren):
  return (fahren - 32) * 5.0/9.0

#method to convert Celsius to Fahrenheit   
def celToFahren(cel):
  return 9.0/5.0 * cel + 32
