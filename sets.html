Sets


Sets, like lists and tuples, are another data type that allows us to store multiple values, just like a list and tuple. However, Sets are a bit different, they are unordered; meaning that values inside cannot be reference by a index. They also do not allow for the same values to appear twice in a single set.

A Set is by using double squiggly brackets {} instead of the square brackets [] that are used with lists or normal brackets () used with tuples.


Code Example:


setOfNames = {"Susan", "Kevin", "Frank"}
setOfAges = {23, 54, 56}

print(setOfNames)
print(setOfAges)

Code Output:


# --Output--

{'Susan', 'Kevin', 'Frank'}
{23, 54, 56}

Using For Loops

We can use for Loops to iterate through every element in Tuple.

Code Example:


setOfNames = {"Susan", "Kevin", "Frank", "Tom", "Ashe", "Bill", "Slim", "Shady"}
setOfAges = {23, 54, 56, 35, 36, 21, 46, 75, 32, 46}

for i in setOfNames:

Remember using for loops with range will not work as we cannot index a set.

Code Example:


setOfNames = {"Susan", "Kevin", "Frank", "Tom", "Ashe", "Bill", "Slim", "Shady"}

setSize1 = len(setOfNames) # len() is a method that returns the size of a tuple

for i in range(setSize1):
  print(setOfNames[i])

Code Output:


# --Output--

line 8, in &itmodule> print(setOfNames[i]) TypeError: 'set' object does not support indexing

Adding and Removing Items

Although we can edit items, we are able to add item to a set, to do this we can use a method called add().

Code Example:


setOfNames = {"Susan", "Kevin", "Frank", "Tom", "Ashe", "Bill", "Slim", "Shady"}

setOfNames.add("Hancock")
print(setOfNames)

Code Output:


# --Output--

{'Susan', 'Ashe', 'Shady', 'Slim', 'Kevin', 'Hancock', 'Bill', 'Tom', 'Frank'}

Note: Notice how ‘Hancock’ was not added to the end of the set, this is because there is no need as there are not indexes.

Joining Sets

Code Example:


setOfNames = {"Susan", "Kevin", "Slim", "Shady"}
setOfAges = {23, 54, 56, 35, 36, 21, 46, 75, 32, 46}
listOfCars = ["Honda", "BMW", "VW", "Toyota"]

setOfNames.update(setOfAges)
setOfAges.update(listOfCars)

print(setOfNames)
print("")
print(setOfAges)

Code Output:


# --Output--

{32, 'Susan', 35, 36, 'Shady', 75, 46, 'Slim', 23, 21, 'Kevin', 54, 56}

{32, 'Honda', 35, 36, 'VW', 75, 46, 'Toyota', 'BMW', 21, 54, 23, 56}

Note: notice how we can update a set with a list too

Code Example:


setOfNames = {"Susan", "Kevin", "Slim", "Shady"}
setOfAges = {23, 54, 56, 35, 36, 21, 46, 75, 32, 46}

setOfNamesAndAges = setOfNames.union(setOfAges)
print(setOfNamesAndAges)

Code Output:


# --Output--

{'Slim', 'Shady', 32, 35, 36, 75, 'Kevin', 46, 21, 54, 23, 56, 'Susan'}
