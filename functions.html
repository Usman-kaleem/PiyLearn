Functions


Say we want to write multiple lines of code, and we want to use this code multiple times. Instead of rewriting these lines of code several times, we can use somethings called a Function. We can now write out code and define it to a function, now every time we want to run this code we call the function.

Defining our Function

Code Example:


print("Hello this line of code is not part a Function\n")

# This is where our function starts
def Our_Function():
  print("Hello this is from code that is in a function")
  a = 1
  b = 2
  print(a + b)
  print("")

Code Output:


# --Output--

Hello this line of code is not part a Function

From the code above we have made our function. However, notice that the code within our function did not run, this is because we have not called our function, we have just defined it.

Using Parameters in a Function

Notice how a function has parameters “()” at the end. We can use these parameters to input Values into our functions. Let us see how we can do this.p>

Code Example:


# We put a variable into these parameters, this variable can be used in #the function 
def WhatsYourName(name):
  print("Hello, my name is " + name)

# When we call the function put a value in the parameters
WhatsYourName("Tommy")

Code Output:


# --Output--

Hello, my name is Tommy

We can add multiple variables (or arguments) within a single parameter.

Code Example:


# Here we are adding multiple arguments and separating them with commas
def NumberCalculation(num1, num2, num3):
  print((num1 + num2) / num3)

NumberCalculation(10,7,2)   # Outputs (10 + 7)/ 2
NumberCalculation(20,10)  # This one is missing an value, we need 3 but only have 2

Code Output:


# --Output--

TypeError: NumberCalculation() takes exactly 3 arguments (2 given) at ⁢unknown>
Note:

Notice how we get an error because the number of arguments needed and the number of arguments given don’t match up.
