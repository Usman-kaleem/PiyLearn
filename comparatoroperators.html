Comparator Operators


We can use comparison operators to compare values in python.

Equals and Not Equals

To see if two values are equal we can use this equals comparator operator. To use it in python is two equal symbols “==”, we can also use “!=” to see if a value is not equal to another. If a value is equal to another it will return True if it is not equal it will return False.

Code Example:


a = 10
b = 5
c = 10

print (a == b)
print (a == c)
print ("")
print (a != b)
print (a != c)

Code Output:


# --Output--

False
True

True
False

Greater Than and Less Then

To see if a values is either bigger or smaller that another we can use the Greater Then operator “>” or the Less Then operator “< ”. Depending on the operator used it will return True or False.

Code Example:


a = 10
b = 5
c = 10

print (a > b)
print (a < b)
print (a > c) # 10 is not bigger then 10 so this will be false
Print (a > c) # 10 is not smaller then 10 so this will be false

Logic Operators

A logic operator is used when we want to compare some True or False statements.

“And” is used if we want to make sure all conditions meet a condition. For example: if we wanted to check if a variable with the value of 10 is both bigger then 10 AND 13.

“Or” is used if we want to make sure at least one condition is met. For example: if we wanted to check if a variable with the value of 10 is both bigger then 10 OR 13.
