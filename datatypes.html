Datatypes


When storing data to a variable this data that is stored can be of different types. For example: text, whole numbers, decimal numbers or even data that only has a value of true and false. This in Python (and all other languages) is called Datatypes.


Code Example:


# Here we are stating the variable of differing data types
name = “Usman”
age = 20
temperature = 24.5

print(type(name))
print(type(age))
print(type(temperature))

// -- Output --

Code Output:


# --Output--

<class 'str'>
<class 'int'>
<class 'float'>

We also have lists tuples and dictionaries.


Code Example:


# Here we are stating the variable of differing data types
