Variables


In python, and almost all other languages, we can store a value or piece of data to a something called a variable. These variables can then be called later in the code.


Code Example:


name = "Usman"
age = 20
eyeColor = "Brown"

print(name)
print(age)
print(eyeColor)

Code Output:


# --Output--

Usman
20
Brown
Note:

Note: Notice how there is no output. That is because we only set a variable but have not called it.

After these variables have been stated or set, we can then call these variable.

Code Example:


# Here we are stating the variable 
name = “Usman”
age = 20
eyeColor = “Brown”
