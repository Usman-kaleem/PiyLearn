Conditions


Previously we learned about operators, these operators returned either true or false. Now let’s see how we can use these true and false statements to run a block of code. We will be learning about, If statements, If Else statements, While Loops and For Loops

If Statements

If we want to run a block of code only if a statements, we give meets a certain criteria (True) we use a if statement.

Code Example:


if True:
  print("This will print because the If statement is true")
  
if False:
  print("This will not print because the If statement is not true")

Code Output:


# --Output--

This will print because the If statement is true
Note:

As seen above any line of code that is indented after the if statement will be execute only if the condition it true. Let’s try another example.

Code Example:


age1 = 12
age2 = 18

if (age1 >= 18):
  print("You are " + str(age1) + " years old. Old enough to vote")

if (age2 >= 18):
  print("You are " + str(age2) + " years old. Old enough to vote")

Code Output:


# --Output--

You are 18 years old. Old enough to vote

Only the second if statement ran the line of code, this is because its condition was true. The variable age2 is greater or equal to 18, whilst age1 is not greater than or equal to 18.

Note:

Note: If you don’t remember what the str() method is, check out the concatenation under the strings tutorial we learned previously. All it does is convert an int into a string so we can concatenate.

If Else and Elif Statements

The If elif statement works exactly the same a if statement but has a second statement that only runs if the last statement was not true. An Else runs if all statements that proceed it are false.

Lets see how a Elif Statement works

Code Example:


if False:
  print("If Statement 1 (if statement)")
  
elif True:
  print("If statement 2 (elif statement)")

else:
  print("If statement 3 (else statement)")

Code Output:


# --Output--

If statement 2 (elif statement)

While Loop

A while loop will run a block of code again and again for as long as a statement is True. This block can sometimes run forever if you are not careful however.

Code Example:


number = 10
while number &it 150:
  print("still in the while loop" + str(number))
  number = number + 10
  
print("while loop has ended")

Every loop we add 10 to number until number is no longer smaller then 150.

Break

We can use Break to leave a loop even if the condition has not yet been yet.

Code Example:


number = 10
while number &it 150:
  print("still in the while loop" + str(number))
  number = number + 10
  break
  
print("while loop has ended")

Code Output:


# --Output--

still in the while loop10
while loop has ended

From the code above notice that as the loop started it printed out a statement then added ten to the variable number but before it could loop back round it was stopped by the break statement.

Continue

We can use Continue to go back to the start of the loop even if there is code that is still beneath.

Code Example:


number = 10
while number &it 150:
  print("still in the while loop" + str(number))
  number = number + 20
  continue
  number = number + 5
  
print("while loop has ended")

Code Output:


# --Output--

# -- Output --

still in the while loop10
still in the while loop30
still in the while loop50
still in the while loop70
still in the while loop90
still in the while loop110
still in the while loop130
while loop has ended

From the code above notice how the variable did not increment by 5, this is due to the continue statement that starts the loop from the beginning.

For Range loops

A for loop using the range() Function Allows us to loop through a block of code a set amount of times. The function adds 1 from 0 to a specific number, for example; if 10 is entered then the functions runs 10 times.

Code Example:


# here we set a normal variable
name = "tommy" 

# This is where we start our for loop. Don’t get confused with the 
letter # “i” this is just another variable.
for i in range(10):
    print(name)
    print(i)
