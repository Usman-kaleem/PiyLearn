Putting our Code Together


We have all the parts needed to make our program. Now let us see what our code looks like when it is all put together. You can play around with the program here.

Complete Code:


def add(num1, num2):
    return num1 + num2

def subtract(num1, num2):
    return num1 - num2

def multiply(num1, num2):
    return num1 * num2

def divide(num1, num2):
    return num1 / num2

def modulus(num1, num2):
  return num1 % num2
  
def floorDivision(num1, num2):
  return num1 // num2

def circleArea(radi):
  pi = 3.14
  return pi * (radi * radi);
  
def circlePerim(radi):
  pi = 3.14
  return 2 * pi * radi;
  
def fahrenToCel(fahren):
  return (fahren - 32) * 5.0/9.0
  
def celToFahren(cel):
  return 9.0/5.0 * cel + 32

def powers(num1, num2):
    return num1 ** num2
