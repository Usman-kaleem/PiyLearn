Comments


In Python we use Comments to add notes into our code. These notes are not executed by python and are only for humans to read. We use the ‘#’ symbol when we want to add a comment.


Code Example:


# This is a comment 
Print(“Hi, my name is”) 
Print(“Chika-Chika”)
Print(“Slim Shady”)
# These three lines of code print three statements

We also have multiline comments. To do this sandwich your comment in between 3 speech marks (“””).


"""  This is a comment 
These three lines of code 
print three statements """

Print(“Hi, my name is”) 
Print(“Chika-Chika”)
Print(“Slim Shady”)

We can also used comments to comment out lines of code, meaning these will not run.


#Print(“Hi, my name is”) 
#Print(“Chika-Chika”)
Print(“Slim Shady”)
